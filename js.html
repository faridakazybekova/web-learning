JavaScript – тест

JavaScript — язык программирования для интерактивности на веб-страницах: обработка событий, анимации, динамическое изменение контента.

Основные возможности JavaScript:

  • Работа с переменными: var, let, const
  • Функции для выполнения действий
  • Обработка событий: onclick, onmouseover и др.
  • Манипуляции с DOM: document.getElementById, querySelector и т.д.
  • Циклы и условия: for, while, if, switch
  • Массивы и объекты
