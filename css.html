CSS (Cascading Style Sheets) — язык стилей, который управляет внешним видом HTML-страниц: цветами, шрифтами, отступами, расположением элементов.

Основные возможности CSS:

  • color — цвет текста
  • background-color — цвет фона
  • font-size — размер шрифта
  • margin, padding — отступы вокруг и внутри элементов
  • border — рамки вокруг элементов
  • display, position — расположение элементов на странице
  • flex, grid — современные системы верстки
